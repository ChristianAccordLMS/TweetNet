Result_Type
(optional) - Specifies what type of search results you would prefer to receive.

The current default is "mixed." Valid values include:

  • mixed: Include both popular and real time results in the response.
  • recent: return only the most recent results in the response
  • popular: return only the most popular results in the response.
Since_ID
(optional) - Returns results with an ID greater than (that is, more recent than) the specified ID.
Until
(optional) - Returns tweets generated before the given date.

Date should be formatted as YYYY-MM-DD.
